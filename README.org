* Futurismoとは
Futurismoとは、20世紀初めにイタリアにおいて勃興した運動。

過去の芸術の徹底破壊と、機械化によって実現された近代社会の速さを称えるアバンギャルド。

- [[http://ja.wikipedia.org/wiki/%E6%9C%AA%E6%9D%A5%E6%B4%BE][未来派 - Wikipedia]]

そんな彼らのちょっとキケンな思想に我も続こう。ここにハッカー未来派宣言を記す。
