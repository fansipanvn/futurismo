** 選定する
選ぶに当たって重視する機能は、

- JUnit連携機能
- リファクタリング
- コード補完・タグジャンプ
- Android開発連携

JUnit連携はmalabarができる。
リファクタリングはどれもいまいちなので諦める。
コード補完やタグジャンプはどれもできる。

ということで、malabar-modeを試すことにした。
