\section*{org-mode形式 -> ODT形式 -> ODP形式 -> PPT形式}
\label{sec-2}
\subsection*{org-mode から ODT形式に変換}
\label{sec-2-1}
ox-odtパッケージを利用する. 
\begin{itemize}
\item \url{https://github.com/emacsmirror/org/blob/master/lisp/ox-odt.el}
\end{itemize}

これは、org-modeにデフォルトで入っているので、
以下のようにrequireする.

\begin{verbatim}
(require 'ox-odt)
\end{verbatim}

実行のためには、

\begin{itemize}
\item zip
\item unzip
\end{itemize}

コマンドに環境変数PATHが通っている必要があった.
自分の場合は、cygwinからダウンロードしたコマンドにパスを通した.

M-x org-odt-export-to-odt を実行すれば、odt形式のファイルが作成される.

\subsection*{ODT形式から PPT形式に変換}
\label{sec-2-2}
LibreOffice Writer からODTファイルを開く.

ファイル -> 送る -> プレゼンテーションに自動抽出

を選択.ODP形式でパワーポイントに出力される. それを PPTとして保存.
\begin{itemize}
\item \href{http://www.libreoffice.org/}{Home | LibreOffice - Free Office Suite - Fun Project - Fantastic People}
\end{itemize}

これでいけた！と思いきや\ldots{} アウトラインしか出力されず、中身空っぽ..
これでは使えない(\#ﾟДﾟ)y-\textasciitilde{}~

\section*{org-mode形式 -> beamer形式 -> PDF形式 -> ODP形式 -> PPT形式}
\label{sec-3}
ox-beamerパッケージを利用する. これは、org-modeにデフォルトで入っているので、
以下のようにrequireする.

\begin{verbatim}
(require 'ox-beamer)
\end{verbatim}

実行のためには、 \texttt{pdflatex} コマンドに環境変数PATHが通っている必要があった.
自分の場合は、cygwinからダウンロードしたコマンドにパスを通した.
\begin{itemize}
\item \href{http://did2memo.net/2014/03/06/easy-latex-install-windows-8-2014-03/}{簡単\LaTeX{}インストールWindows編（2014年7月版）}
\end{itemize}

M-x org-odt-export-to-odt を実行すれば、odt形式のファイルが作成される.
